                        üöÄ HOW TO USE
================================================================================

1. LOGIN TO BOING
   - Go to: http://localhost:5173
   - Email: admin@example.com
   - Password: admin123

2. TRY SETTINGS
   - Click "Settings" button
   - Update your profile
   - Change password if you want

3. SETUP DEMO API
   - cd demo-api
   - pip install -r requirements.txt
   - python app.py

4. REGISTER IN BOING
   - Go to "APIs" page
   - Click "+ Add API"
   - Name: "Demo E-commerce API"
   - Copy the API key

5. CONFIGURE DEMO API
   - Open demo-api/app.py
   - Set: BOING_API_KEY = "your-key-here"
   - Restart: python app.py

6. GENERATE TRAFFIC
   - Open new terminal
   - cd demo-api
   - python test_api.py
   - Press Enter

7. WATCH THE MAGIC
   - Go to Boing Dashboard
   - See live requests
   - Check Alerts page for threats
   - View Logs for details
